# İçerix - Kullanıcı Akışı

1. Kullanıcı icerix.netlify.app adresine girer
2. Ürünün markasını, adını veya barkodunu giriş kutusuna yazar
3. "Analiz Et" butonuna tıklar
4. Yapay zeka içeriği analiz eder
5. İki sonuç ekranı gösterilir:
   - 1. Kart: Sağlık skoru + riskli maddeler (kırmızı), dikkat edilecek maddeler (turuncu), güvenli maddeler (yeşil)
   - 2. Kart: Sağlıklı alternatif önerileri
6. Kullanıcı sonuçları okuyarak bilinçli kararını verir
