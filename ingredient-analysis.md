# Temel Özellik: İçerik Analizi

## Özellik Açıklaması
Kullanıcı ürün adını girer, Grok API içerik listesini analiz eder ve sonuçları iki kart halinde sunar.

## Kullanıcı Girdisi
- Ürün markası, adı veya barkod numarası

## AI Çıktısı
- Kart 1: Sağlık skoru + kırmızı/turuncu/yeşil renk kodlu içerik analizi
- Kart 2: Sağlıklı alternatif önerileri

## Kullanılan Teknoloji
- Grok API
- HTML5, CSS3, JavaScript
